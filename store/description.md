# ChatGPT Manager - Chrome Web Store Listing

## Short Description (132 characters max)
Manage your ChatGPT conversations effortlessly. List, preview, search, and delete chats without refreshing the page.

## Detailed Description

### Overview
ChatGPT Manager is a lightweight Chrome extension that gives you complete control over your ChatGPT conversations. No more scrolling through endless chat history or accidentally losing important conversations.

### Key Features

**Quick Conversation Overview**
- See all your ChatGPT conversations in a clean, organized list
- Preview conversation content with a single click
- View message snippets without opening each chat

**Powerful Search**
- Instantly search through all your conversations
- Filter by conversation title or content
- Find any chat in seconds

**Safe Deletion**
- Delete conversations with confirmation dialog
- Optional backup before deletion
- Batch delete multiple conversations at once

**Smart Sync**
- Automatic background synchronization
- Cache-first loading for instant access
- Works even when offline (with cached data)

### Why ChatGPT Manager?

- **Non-intrusive**: Never refreshes your ChatGPT page or logs you out
- **Privacy-focused**: All data stays in your browser
- **Fast & Lightweight**: Minimal resource usage
- **Beautiful UI**: Clean, modern design inspired by professional tools

### How It Works

1. Install the extension
2. Open ChatGPT in your browser
3. Click the extension icon to manage your conversations
4. Search, preview, and delete with ease

### Permissions Explained

- **tabs**: To communicate with your ChatGPT tab
- **storage**: To cache conversations locally for faster access
- **activeTab**: To read conversation data from ChatGPT
